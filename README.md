# ShelfScry

A book discovery app. Describe a plot or theme in plain English — ShelfScry finds real matching books.

## Prerequisites

- Node.js 18+
- A Groq api skey

## Setup

1. `npm install`
2. Copy `.env.example` to `.env` and paste your Groq API key
3. `npm run dev`
4. Open `http://localhost:3000`

## Usage

Type any plot description and press Enter. Examples:

- "orphan discovers he has magical powers and goes to a school for wizards"
- "dead parents but grows to get revenge as an assassin"
- "woman in a dystopian society where women have no rights"
